# ratcn

ratcn is a component library for [Ratatui](https://ratatui.rs/) apps: beautifully designed terminal UI components that you can copy, paste, theme, and own in your application code.

This repository is currently a placeholder while the library is being built. A preview site is coming soon with live component demos and more information about the direction of the project:

[ratcn.kristoferlund.se](https://ratcn.kristoferlund.se)

If you find this repository before the first release, please check out the preview and open an issue if there are specific components, patterns, or features you would like to see included. Early requests will help shape what gets prioritized before the library is released.
